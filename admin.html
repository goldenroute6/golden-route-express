Admin Booking (Simple Mode)

Booking is managed directly in Supabase Table Editor. Public website is tracking only.

Step 1

  1. Open your Supabase project: nytgmlaiecrmrnymgclm
  2. Go to Table Editor.
  3. Open table: shipments.
  4. Click Insert row.
Open Supabase Dashboard Open Public Tracker

Step 2

Fill these two columns in each row:

  • tracking_number: example GRE-2026-9001
  • payload: paste shipment JSON template

Template file in repo:

shipment-template.json

Step 3

Save row, then test from any phone in public tracker:
